Acerca de

Este proyecto del curso de IF7102- Multimedios tiene como propósito, el de realizar una aplicación web que consuma una API proveniente de la pagina https://www.themoviedb.org, para evidenciar el manejo de los lenguajes de programación tales como, HTML, CSS, JAVASCRIPT, PHP, entre otros. Además, se le presentara un formulario con el fin de facilitarle al usuario un medio de comunicación que le permita al mismo enviar cualquier consulta referente al uso de la pagina Web.
